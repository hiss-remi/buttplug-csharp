Connect(Uri, Boolean)

Creates the connection to the Buttplug Server and performs the protocol handshake.

Declaration
public Task Connect(Uri aURL, bool aIgnoreSSLErrors = false)
Parameters
Type Name Description
Uri aURL

The URL for the Buttplug WebSocket Server, in the form of wss://address:port (wss:// is to ws:// as https:// is to http://)
Boolean aIgnoreSSLErrors

When using SSL (wss://), prevents bad certificates from causing connection failures
Returns
Type Description
Task

Nothing (Task used for async/await)
Remarks

Once the WebSocket connection is open, the RequestServerInfo message is sent; the response is used to set up the ping timer loop. The RequestDeviceList message is also sent, so that any devices the server is already connected to are made known to the client.

Important: Ensure that DeviceAdded, DeviceRemoved and ErrorReceived handlers are set before Connect is called.
